This extension enables advanced comparison of sequences of HTTP requests and their responses. It is particularly useful for analyzing differences and similarities between two sets of HTTP sequences.

Features

  • Context Menu Integration: Send requests from Burp Suite's Proxy tab to the extension via the context menu.
  • Sequence Overview: View sequences with customizable names, number of requests, and total response length. Sequence order can be reversed.
  • Side-by-Side Sequence Display: Display two sequences simultaneously for easy comparison, with separate tables for each sequence.
  • Longest Common Subsequence (LCS) Analysis: Identify the LCS between two sequences using a Dynamic Programming algorithm. Color-coded requests for easy distinction.
  • Request/Response Diff View: View detailed differences in request/response content, with color-coded diff markers for added, modified, and deleted content.
  • Scroll Synchronization: Optional scroll synchronization between the two sequence panels.
  • Auto-Select Mode: Automatically select the corresponding request in the other sequence when one request is selected.

Usage

  1. Select requests from Burp Suite's HTTP history or repeater.
  2. Right-click and choose Send to Sequence Comparer.
  3. Open the Sequence Comparer tab to analyze sequences:
    • Use LCS analysis for comparison.
    • View differences using the diff algorithm.
    • Enable scroll synchronization and auto-select mode as needed.
